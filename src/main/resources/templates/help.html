Help

Voici un récapitulatif des commandes disponibles :

Liste des STB

  • Format XML

    • URL : /stb23/resume/xml
    • Méthode : GET
    • Affiche la liste des spéfications stockés

  • Format HTML

    • URL : /stb23/resume/html
    • Méthode : GET
    • Affiche la liste des spéfications stockés

Détails d'une spéfication

  • Format XML

    • URL : /stb23/resume/xml/{id} {id} = valeur numérique de l’identifiant
    • Méthode : GET
    • Affiche le contenu complet de la spéfication dont l’identifiant est {id}

  • Format HTML

    • URL : /stb23/html/{id} {id} = valeur numérique de l’identifiant
    • Méthode : GET
    • Affiche le contenu complet de la spéfication dont l’identifiant est {id}

Ajout d'une STB dans la base

  • URL : stb23/insert
  • Méthode : POST
  • Le flux reçu est validé par le schéma XSD de définition stb23.

Suppression d'une spéfication

  • URL : /stb23/delete/{id} {id} = valeur numérique de l’identifiant
  • Méthode : DELETE
  • Suppression de la spéfication dont l’identifiant est {id}
